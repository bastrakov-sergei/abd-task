Разработать веб-приложение, реализующее следующую функциональность:
1. Администратор системы
  1. Войти в систему
  2. Создать новую учетную запись администратора
  3. Удалить учетную запись администратора
  4. Загрузить файл со списком точек через веб интерфейс. Если точка с указанным идентификатором уже есть, надо отредактировать существующую точку, если точки нет, то добавить новую.
  5. Добавить новую точку через веб-интерфейс
  6. Отредактировать существующую точку через веб-интерфейс
  7. Удалить точку через веб-интерфейс
2. Клиент
  1. Возможность по API запросить список точек с данными, находящимися поблизости к определённой координате
 
Есть 3 типа файлов, в которых находятся данные по точкам: банкоматы, офисы, инфо-киоски. Структура и формат этих файлов различный. Идентификаторы между разными типами файлов различные, и могут пересекаться, поэтому надо различать, из какого типа файла была получена указанная точка и сохранять их как разные сущности.

Необходимо использовать один из фреймворков ASP.NET MVC  или .NET Core MVC, в качестве хранилища данных использовать MS SQL. Клиентская часть может быть реализована с помощью bootstrap либо на усмотрение выполняющего тестовое
